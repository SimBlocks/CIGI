# Overview
Common Image Generator Library Middleware and Tools

# Setup 
1. Initialize Submodules
a. Open a command window
b. Navigate to the cloned CIGI folder
c. git submodule update --init --recursive
d. Wait for the submodules to finish downloading

2. Follow instructions in thirdparty/README.md
3. Follow instructions in simulationsdk/README.md
